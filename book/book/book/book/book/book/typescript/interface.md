# 接口

##### 接口的作用
接口定义的是一种标准，在实现的时候，必须遵循此接口定义的标准。接口用interface关键字定义。
##### 接口的分类
1. 属性接口
2. 函数接口
3. 可索引接口
4. 类类型接口
5. 接口扩展
