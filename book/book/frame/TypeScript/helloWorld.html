6.2.2 环境搭建

环境安装

全局安装ts

npm install typescript -g

查看ts版本

tsc --version

生成package.json文件

npm init -y

ts配置文件生成（编译成js的规则）

tsc --init

types/node包生成(解决模块的声明文件问题)

npm i @types/node --save-dev

编写helloWorld.ts

构建

vscode编辑器---->运行任务---->tsc：监听-tsconfig.js,会生成一个helloworld.js文件，并自动编译成js文件。

运行

在html中引入的时候，引入编译后的js文件，就可以在浏览器中运行了。
